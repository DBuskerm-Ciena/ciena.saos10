***********************
ciena.saos10.saos10_fps
***********************

**Manage forwarding domains on Ciena SAOS 10 devices**


Version added: 1.5.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- This module provides declarative management of a forwarding domain



Requirements
------------
The below requirements are needed on the host that executes this module.

- ncclient (>=v0.6.4)
- xmltodict (>=0.12.0)

Examples
--------

.. code-block:: yaml

    # Using merged

    - name: Configure interfaces
      ciena.saos10.saos10_fps:
        config:
          - name: fp1
            fd-name: foo
            logical-port: 1
        state: merged


    # Using overridden

    - name: Configure interfaces
      ciena.saos10.saos10_fps:
        config:
          - name: fp1
            fd-name: foo
            logical-port: 1
        state: overridden


    # Using deleted

    - name: Configure interfaces
      ciena.saos10.saos10_fps:
        config:
          - name: fp1
        state: deleted
